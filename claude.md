# Claude / AI アシスタント向けメモ

## このプロジェクトの目的

**親子が一緒に楽しみながら、フットボールの戦術ボード（作戦板）を作れること**を第一目的とする。

- 子どもが「選手を動かす」「顔写真を載せる」「名前を自分の呼び方に変える」など、**遊びと参加感**を通じてピッチ上のイメージをつかむ。
- 保護者やコーチが、**守備・攻撃のシーン切り替え**や**矢印（パス・走り込み）**で説明しやすくする。
- **保存と履歴**で、練習前後の振り返りや家族での会話の材料にする。

実装・ドキュメント・UI 文言を変更するときは、**専門用語に寄りすぎず**、子どもにも視覚的に分かりやすい体験を損なわない方向を優先する。

## 技術の要点（短縮）

- メイン UI: `src/tactics-board_1.jsx`（React + Tailwind + lucide-react）。
- 保存: `window.storage`（ブラウザでは `src/storage-polyfill.js` が `localStorage` に橋渡し）。
